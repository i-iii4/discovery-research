# Профиль основателя: saas (исходный)

Базовый профиль, под который калибровались пайплайны v1–v15. Web/SaaS, solo-разработчик.

- **Стек:** full-stack, любая платформа
- **Области интереса:** Productivity, PKM, AI/LLM, Health/Wellness, Creator tools, SaaS tools
- **Anti-области:** iGaming
- **Целевой доход:** $5K+/мес MRR
- **Ограничение:** 2 недели до MVP, один разработчик
- **Дистрибуция:** не формализована — главный фильтр пайплайна это спрос (demand), а не дистрибуция

Атрибут в артефактах: `profile: saas`.
